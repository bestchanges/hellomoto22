
В данной директории можно размещать пользовательские скрипты:

before_reboot_temp.bat
	Будет запускаться при нарушении температуры при установленной опции "авторебут при нарушении температуры".
	Если на следующей итерации проблема температуры не решена, то система будет перезагружена.

before_reboot_card.bat
	Будет запускаться при отвале видеокарт при установленной опции "авторебут при отвале видеокарт".
	Если на следующей итерации видеокарты не восстановлены, то система будет перезагружена.
