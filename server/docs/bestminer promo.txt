== How it works.

Each GPU mining hardware has different hashrate for each mining algorithm.
So first of all BestMiner starts with benchmarking your rig of each possible algorithm.

Profit of mining depends on several parameters, including: network hashrate, difficulty, block time, pool luck, exchange rate and others.
BestMiner continuously monitoring these parameters and choose the most profitable combination of altcoin/pool/exchange.
When it run corresponding mining algorithm.

After receiving mining revenue from Pool BestMiner converts it to BTC and send to your wallet.
