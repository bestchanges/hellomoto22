ТЗ на сайт allminer.

Работа на стороне клиента
запуск:
1. обновить конфиг
2. если есть обновления программы, обновиться
3. запустить майнер (если не запущен)
2. получить задачи, выполнить задачи
4. передать статистику

Сущности:
* пользователь
  * логин-пароль
  * ключ
* настройки монет - какие монеты и на каких пулах работают
  * [монета-пул-сервер-аккаунт_пула]
* группы ригов
  * майнер
  * конфигурация майнера
* риги
  * риг id (CPU ID)
  * риг name
  * аккаунт->
  * группа->
  * настройка монет
  * заметка
  * целевые хешрейты
    [алгоритм-хешрейт_макс-хешрейт_падение(?)]
  * разгон
    * алгоритм
    * мощность-память-ядро-даунвольт
  * карты
    * название
    * тип (nvidia/amd)
    * память объем
    
* история хешрейта
  * период (часы)
  * риг->
  * монета
  * пул->
  * значение
  * вид (декларируемый, фактический, расчетный)
* алерты
  * риг->
  * уровень (желтый, красный, коричневый)
  * сколько давно  
* перезагрузки
  * риг-
  * дата выключения
  * дата включения
  * период офлайн
* прибыльность фактическая
  * риг-
  * дата (день)
  * число шар
  * монетаc
  * заработано монет
  * валюта клиента


* курсы
  * дата (час)
  * монета-монета
  * курс
  * источник (биржа, индекс)
  * объем торгов
  
  справочники
* монеты
  * монета
  * алгоритм
  * текущая сложность
  * текущий хэшрейт
  * доступные майнеры?
    * майнер
    * конфигурация

* пулы
  * монета->
  * пул
  * [сервера]
  * хэшрейт текущий

* майнеры
  * [алгоритмы]
  * где работает (виндоус-линукс)
  * на каких картах (amd-nvidia-оба)
  * конфигурация майнера
