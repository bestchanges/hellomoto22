Here you can download BestMiner client program which is need to be installed in each rig farm.

  1. Download ZIP file to your Rig Farm
  2. create directory 'bestminer' (or any other) and unZIP file in this directory
  3. run BestMiner.bat
  4. the miner is run. Wait for ~20 seconds and you will see it on my workers page
  5. add BestMiner.bat to system startup

Note: You can install the same ZIP file on every rig. You do not need to download it each time separately.
